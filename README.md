# 📋 Task Manager

Aplicação de lista de tarefas com persistência de dados no navegador.

## 🚀 Funcionalidades
- Adicionar tarefas
- Marcar como concluída
- Excluir tarefas
- Filtro por status
- Salvamento automático (LocalStorage)

## 🛠 Tecnologias
- HTML
- CSS
- JavaScript

## 💾 Armazenamento
Dados salvos no LocalStorage do navegador
